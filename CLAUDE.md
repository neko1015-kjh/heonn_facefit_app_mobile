# FaceFit (모바일 앱)

얼굴 관련 기능을 제공하는 모바일 앱입니다.

## 작업 규칙 (반드시 지킬 것)

- **모든 응답과 코드 주석은 한국어로** 작성합니다.
- **사용자는 비개발자**입니다. 전문 용어를 줄이고, 변경한 내용을 쉽고 친절하게 설명합니다.
- **한 번에 한 가지 변경만** 적용합니다. 여러 가지를 동시에 바꾸지 않습니다.
- **새 라이브러리(패키지)를 추가하기 전에 반드시 사용자에게 먼저 묻고** 동의를 받습니다.

## 기술 스택

- **Expo** (SDK 최신 — 현재 SDK 56)
- **React Native**
- **TypeScript**
- 웹 미리보기: `react-native-web`, `react-dom`, `@expo/metro-runtime`

## 백엔드

- 백엔드는 이 폴더가 아닌 **별도 폴더(`../heonn_facefit_app_backend`)** 에서 작업합니다.
- 백엔드는 **Python + FastAPI** 서버이며, 앱은 이 서버와 통신합니다.

## 실행 방법

- 웹 브라우저 미리보기: `npm run web`
- 모바일(Expo Go 앱): `npm start` 후 QR 코드 스캔

## 참고

- Expo는 버전마다 사용법이 달라집니다. 코드를 작성하기 전에
  SDK 56 공식 문서(https://docs.expo.dev/versions/v56.0.0/)를 확인하세요.
